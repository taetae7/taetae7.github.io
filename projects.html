Developer for RGBudget (Currently in development)

function: users can input daily expenses and keep track of their spending

Code Name: RGBudget (App Icon coming soon) is a simple application where the user can set a monthly budget and see their spending with just a glance. The objective of this application is for its simplicity: users will easily be able to navigate and get the information. On the main screen, the app lets the user know how much money they have left to spend for remainder of the month. The user can click on any date in the calendar and add a description and amount. The user can then click on the circle to the left to indicate a "loss" or "gain" and the app will automatically calculate the net total.
